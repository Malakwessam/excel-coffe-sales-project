# excel-coffe-sales-project
This project analyzes coffee sales data using Microsoft Excel

Coffee Sales Dashboard (Excel)
Overview

This project presents an interactive Coffee Sales Dashboard built using Microsoft Excel. The objective of the analysis is to explore sales performance across different products, regions, and time periods to identify key business insights.

Dataset:
The dataset was obtained from Kaggle and includes information about coffee orders, customers, sales values, and product types.

Process:
Data cleaning and preparation in Excel
Data transformation for analysis
Creation of Pivot Tables for summarizing data
Development of an interactive dashboard using charts and slicers

Key Insights:
Comparison of sales across different countries
Sales performance by coffee type
Monthly sales trends over time
Identification of top customers

Tools Used:
Microsoft Excel
Pivot Tables
Charts and Slicers
